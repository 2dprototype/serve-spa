# serve-spa

A CLI tool to quickly serve Single Page Applications with automatic QR code generation for easy mobile testing.

## Features

- 🚀 Quick SPA serving with zero configuration
- 📱 Automatic QR code generation for mobile testing
- 🌐 Auto-detection of network IP address
- 📁 Automatic static directory detection
- 🔒 CORS support when needed
- 🎯 Custom base path support
- 🖥️ Auto-opens browser (optional)
- 🛑 Graceful shutdown

## Installation

### Global Installation

```bash
npm install -g serve-spa
